# policies — тексты политик по языкам

Кладётся в репозиторий Qurb-content, папка `policies/`.
Приложение подгружает `policies/<kind>_<lang>.md.gz` по кнопке «Читать полностью».

kind: `privacy` (конфиденциальность) | `shariah` (соответствие шариату).
Языки: en, ru, ar, kk, ky, az, uz, tg, tr, ja.

ВАЖНО:
- Privacy — ШАБЛОН. Адаптируйте под реальные практики приложения, замените email и дату.
  Для Google Play политика должна отражать фактическую обработку данных.
- Shariah — вычитайте с учётом местной терминологии/мазхаба.
