# geo/cities — города мира на 10 языках (для глобуса и поиска)

Дополнение к репозиторию Qurb-content. Кладётся в папку `geo/` рядом с
`countries.json.gz`. Раздаётся через jsDelivr, качается приложением и кэшируется.

## Файлы

- `geo/cities.json.gz` — **15 423 города** мира: координаты, население, признак
  столицы и названия на 10 языках. 99% городов переведены. Размер ~0.9 МБ.
- `geo/cities_index.json` — манифест (кол-во, языки, формат).
- `geo/by_country/` — пусто. Сюда позже добавляются детальные города по странам
  (`KZ.json.gz`, `RU.json.gz` …), когда захочешь опуститься ниже 15 тыс. населения.

## Формат cities.json.gz (после распаковки)

Массив, отсортированный по населению по убыванию (крупные города первыми — удобно
для отрисовки глобуса и прореживания по зуму):

```json
[
  {
    "name": "Almaty",           // английский оригинал (это же язык en)
    "country": "Kazakhstan",
    "lat": 43.2567, "lon": 76.9286,
    "pop": 2000900,
    "capital": false,
    "tr": {                      // переводы названия; отсутствует => бери name
      "ru":"Алматы","ar":"…","kk":"Алматы","ky":"…",
      "az":"…","uz":"…","tg":"…","tr":"…","ja":"アルマトイ"
    }
  }
]
```

Языки (10): en (= `name`), ru, ar, kk, ky, az, uz, tg, tr, ja.
Правило показа: `tr[язык]` если есть, иначе `name`.

## Как расширять по странам (на будущее)

Инструментом `tools/add_cities.py` добавляешь города одной страны в исходные
данные, пересобираешь пак и кладёшь `geo/by_country/<ISO>.json.gz`. Приложение
догружает детальную базу страны только когда пользователь её открывает.
